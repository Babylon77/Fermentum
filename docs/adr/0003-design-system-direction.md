# ADR 0003: Design System Direction

## Status

Accepted

## Context

Most open-source brewing tools are functionally deep but visually weak. Fermentum needs a deliberate, premium, recognizable interface that works across mobile and desktop.

## Decision

Adopt a cyberpunk mission-command design system:

- dark operational surfaces
- high-contrast status colors
- dense but readable information panels
- tactical dashboards, not consumer-card minimalism
- strict design tokens for color, spacing, radius, and component states

## Consequences

Positive:

- strong visual identity
- consistent UI implementation
- suitable for data-heavy brewing workflows

Tradeoffs:

- requires discipline to avoid visual noise
- charting and glow effects must stay restrained
- accessibility needs active review to maintain contrast and readability
