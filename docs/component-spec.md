# Fermentum Component Specification

## Purpose

Translate the style guide into implementation-level component rules.

## Initial components

- app shell
- side navigation
- top command bar
- panel card
- metric tile
- text input
- select input
- segmented control
- primary button
- secondary button
- danger button
- data table
- timeline step
- chart container
- AI response panel

## Shared requirements

- dark mission-command styling
- keyboard and touch accessibility
- visible focus states
- consistent spacing and radius tokens
- disabled and error states defined

## Priority implementation order

1. app shell
2. panel card
3. buttons
4. inputs
5. metric tile
6. data table
7. timeline step
8. AI response panel
